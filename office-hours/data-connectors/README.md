# data-connectors

## Office Hours: Connect Once, Query Everywhere

### Description

This 30 minute Office Hours interactive & hands-on webinar focuses on the various data connectors available for Trino and Starburst. Discussion centers on the different types of connectors, how to configure them, and how to use them to query data from a variety of sources -- including federated queries across multiple data sources.
